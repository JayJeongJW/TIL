## EVE 온라인 세계관과 인게임 시스템 연결 정리

## 오늘 조사한 내용

오늘은 EVE 온라인의 세계관이 어떤 방식으로 구성되어 있고, 그 배경 설정이 실제 게임 시스템과 어떻게 연결되는지 조사했다.

EVE 온라인은 단순히 우주를 배경으로 한 MMORPG가 아니라, 세계관 자체가 게임의 경제, 전쟁, 이동, 사망, 조직, PvP 구조와 강하게 연결되어 있는 게임이었다.

---

## 1. EVE 온라인의 배경 설정

EVE 온라인의 배경은 `뉴 에덴`이라는 우주 성단이다.

인류는 과거 `EVE Gate`라는 거대한 통로를 통해 뉴 에덴에 도달했다.
하지만 이후 EVE Gate가 붕괴하면서 지구와의 연결이 끊어졌고, 뉴 에덴에 남은 인류는 오랜 암흑기를 겪었다.

이후 살아남은 인류 문명은 각자의 방식으로 다시 성장했고, 현재는 여러 제국과 세력이 공존하는 우주 사회가 되었다.

뉴 에덴의 주요 세력은 다음과 같다.

| 세력       | 특징                                                  |
| -------- | --------------------------------------------------- |
| 아마르 제국   | 종교, 황제, 신분 질서, 노예제 이미지가 강한 제국                       |
| 칼다리 합중국  | 거대 기업과 군사 규율 중심의 기업국가                               |
| 갈렌테 연방   | 자유, 민주주의, 문화, 개인주의를 강조하는 연방                         |
| 미마타르 공화국 | 아마르에게 지배당했던 과거를 가진 해방과 저항의 민족                       |
| CONCORD  | 제국 사이의 질서와 치안을 조정하는 조직                              |
| 해적 세력    | Angel Cartel, Blood Raiders, Guristas 등 범죄 또는 독립 세력 |
| 플레이어 조직  | 기업, 동맹, 연합 형태로 실제 우주 정치와 전쟁을 만드는 세력                 |

EVE 온라인의 세계는 선과 악이 명확하게 나뉜 세계라기보다, 제국, 기업, 종교, 해적, 플레이어 조직이 각자의 이익을 위해 움직이는 회색지대에 가깝다.

---

## 2. 플레이어의 정체성: 캡슐리어

EVE 온라인에서 플레이어는 일반 우주선 조종사가 아니라 `캡슐리어`이다.

캡슐리어는 캡슐 안에서 우주선과 직접 연결되어 조종하는 존재이며, 클론 기술을 통해 죽어도 다시 부활할 수 있다.

이 설정은 게임 시스템과 직접 연결된다.

일반적인 게임에서는 캐릭터가 죽어도 다시 부활하는 것이 단순한 시스템 편의일 수 있다.
하지만 EVE 온라인에서는 이것을 세계관 안에서 설명한다.

* 플레이어는 함선이 파괴되어도 캐릭터 자체는 계속 유지된다.
* 캡슐까지 파괴되면 클론으로 다시 깨어난다.
* 죽음은 끝이 아니라 함선, 장비, 자원, 시간의 손실로 처리된다.
* 플레이어는 불멸에 가까운 우주선 조종사로서 계속 전쟁과 경제에 참여한다.

즉, EVE 온라인은 `죽어도 다시 출격하는 MMO 구조`를 `캡슐리어와 클론 기술`이라는 세계관 설정으로 자연스럽게 연결했다.

---

## 3. 배경 설정을 플레이어에게 체험시키는 방식

EVE 온라인은 세계관을 단순히 텍스트나 컷신으로만 보여주지 않는다.

플레이어가 직접 세계관 안에서 행동하게 만든다.

대표적인 체험 방식은 다음과 같다.

| 체험 방식    | 설명                                    |
| -------- | ------------------------------------- |
| 제국 선택    | 캐릭터 생성 시 네 제국 중 하나의 배경을 선택            |
| 우주 이동    | 스타게이트를 통해 성계 사이를 이동                   |
| 보안 등급 체험 | 지역마다 치안 수준과 위험도가 다름                   |
| 함선 손실    | 함선이 파괴되면 실제 자산 손실이 발생                 |
| 기업 가입    | 플레이어 조직에 들어가 경제, 전쟁, 정치에 참여           |
| 제국 전쟁    | Faction Warfare를 통해 특정 제국 편에 서서 전쟁 참여 |
| 시장 거래    | 채굴, 생산, 운송, 거래를 통해 우주 경제 체험           |
| 탐사       | 웜홀, 데이터 사이트, 유물 사이트를 찾아 미지의 공간 탐험     |

이 구조 때문에 EVE 온라인의 세계관은 “읽는 설정”이 아니라 “직접 겪는 설정”에 가깝다.

## 4. 보안 등급과 우주의 위험도

EVE 온라인의 우주는 지역마다 보안 등급이 다르다.

이 보안 등급은 단순 난이도 구분이 아니라, 세계관 속 질서와 무법의 차이를 직접 체험하게 만드는 장치이다.

| 지역             | 특징                               |
| -------------- | -------------------------------- |
| High Security  | CONCORD의 치안이 강하게 작동하는 비교적 안전한 지역 |
| Low Security   | 제국 영향권이지만 위험이 커지는 지역             |
| Null Security  | 법이 거의 없는 무법 지역                   |
| Wormhole Space | 일반 스타게이트 질서 밖에 있는 미지의 공간         |
| Pochven        | 트리글라비안 세력과 관련된 특수 지역             |

High Security에서는 범죄 행위에 대해 치안 세력이 반응하지만, Null Security에서는 플레이어 조직이 사실상 질서를 만든다.

즉, EVE 온라인은 공간 자체가 세계관을 표현한다.

안전한 제국 중심부와 위험한 무법지대가 나뉘고, 플레이어는 어느 공간으로 갈지 직접 선택하게 된다.

---

## 5. 시스템과 세계관이 연결된 부분

EVE 온라인에서 가장 인상적인 점은 게임 시스템이 대부분 세계관과 연결되어 있다는 것이다.

### 5-1. 사망과 클론 시스템

캡슐리어는 클론 기술로 부활할 수 있다.
그래서 플레이어가 죽어도 캐릭터가 사라지지 않고 다시 활동할 수 있다.

이 시스템은 MMO의 부활 구조를 세계관 안에서 자연스럽게 설명한다.

### 5-2. 스타게이트 이동

뉴 에덴은 스타게이트로 연결된 성계들로 구성되어 있다.
플레이어는 스타게이트를 통해 성계 사이를 이동한다.

이동 경로는 물류, 전쟁, 해적 활동과 연결된다.
특정 게이트는 이동 통로이면서 동시에 매복 지점이 된다.

### 5-3. 보안 등급

CONCORD와 제국 치안이 존재하지만 모든 우주를 통제하지는 못한다.
이 설정은 Highsec, Lowsec, Nullsec 같은 보안 등급 시스템으로 구현된다.

### 5-4. 제국과 함선 체계

각 제국의 문화와 성향은 함선 디자인과 전투 방식에도 반영된다.

| 제국   | 전투/함선 이미지                    |
| ---- | ---------------------------- |
| 아마르  | 장갑 방어, 레이저, 종교 제국 이미지        |
| 칼다리  | 실드, 미사일, 군사 기업국가 이미지         |
| 갈렌테  | 드론, 하이브리드 무기, 자유주의 연방 이미지    |
| 미마타르 | 빠르고 거친 함선, 발사체 무기, 저항 세력 이미지 |

### 5-5. 플레이어 기업과 동맹

EVE 온라인의 플레이어 조직은 일반적인 길드보다 훨씬 강한 역할을 한다.

플레이어 기업과 동맹은 실제로 다음과 같은 활동을 한다.

* 자원 채굴
* 함선 생산
* 물류 운영
* 세금 관리
* 영토 방어
* 대규모 전쟁
* 외교 협상
* 스파이 활동
* 배신과 자산 탈취

특히 무법 지역에서는 플레이어 동맹이 성계를 지배하고, 사실상 국가처럼 행동할 수 있다.

### 5-6. 플레이어 주도 경제

EVE 온라인은 플레이어 주도 경제가 강한 게임이다.

플레이어는 자원을 채굴하고, 정제하고, 함선과 장비를 만들고, 시장에서 거래한다.
전쟁이 발생하면 함선이 파괴되고, 다시 생산 수요가 생긴다.

즉, 전쟁과 경제가 서로 연결되어 있다.

```text
채굴
↓
생산
↓
시장 거래
↓
전쟁
↓
함선 파괴
↓
다시 생산 수요 발생
```

이 구조 때문에 EVE 온라인의 우주는 실제 경제가 돌아가는 사회처럼 느껴진다.

## 6. 컨텐츠에 반영된 세계관

EVE 온라인의 여러 컨텐츠는 세계관과 직접 연결되어 있다.

| 컨텐츠             | 세계관 반영                   |
| --------------- | ------------------------ |
| 미션              | 제국, 기업, 해적 세력의 의뢰를 수행    |
| Faction Warfare | 네 제국의 전쟁에 플레이어가 직접 참여    |
| 해적 사이트          | 범죄 조직과 적대 세력을 전투 컨텐츠로 구현 |
| 웜홀 탐사           | 미지의 우주와 고대 기술 탐험         |
| 채굴/생산           | 우주 산업 사회 구현              |
| 시장 거래           | 플레이어 주도 경제 구현            |
| Nullsec 전쟁      | 플레이어 조직이 영토를 차지하고 전쟁 수행  |
| 평판 시스템          | 특정 세력을 도우면 다른 세력과 관계가 변함 |

이처럼 EVE 온라인은 세계관이 배경으로만 존재하지 않고, 실제 컨텐츠와 시스템으로 구현되어 있다.

---

## 7. 조사하면서 느낀 핵심

EVE 온라인은 세계관과 시스템이 강하게 연결된 게임이다.

많은 게임은 세계관을 컷신, 대사, 퀘스트 텍스트로 전달한다.
하지만 EVE 온라인은 플레이어가 직접 세계관 안에서 경제 주체, 전쟁 세력, 탐험가, 해적, 기업인, 용병이 되게 만든다.

특히 인상 깊었던 부분은 다음과 같다.

1. 캡슐리어 설정으로 사망과 부활 시스템을 세계관 안에 자연스럽게 넣었다.
2. 보안 등급으로 질서와 무법이 공존하는 우주를 표현했다.
3. 플레이어 기업과 동맹이 실제 정치 세력처럼 작동한다.
4. 전쟁과 경제가 서로 연결되어 있다.
5. 배경 설정이 단순 설명이 아니라 게임 시스템과 직접 연결되어 있다.

## 오늘 배운 점

오늘 조사를 통해 세계관은 단순히 게임의 배경 이야기가 아니라, 게임 시스템을 설득력 있게 만드는 기준이 될 수 있다는 것을 알게 되었다.

EVE 온라인은 “우주에서 여러 세력이 싸운다”는 설정을 단순히 말로만 설명하지 않는다.
플레이어가 직접 세력을 선택하고, 위험한 지역으로 이동하고, 함선을 잃고, 경제 활동을 하고, 조직에 들어가 전쟁에 참여하게 만든다.

즉, EVE 온라인의 세계관은 다음과 같이 정리할 수 있다.

```text
세계관 설정
↓
게임 시스템
↓
플레이어 행동
↓
새로운 사건 생성
```

이 구조가 EVE 온라인의 가장 큰 특징이라고 느꼈다.

앞으로 다른 게임의 세계관을 분석할 때도 단순히 “무슨 배경인가?”만 보는 것이 아니라, 그 배경이 실제 플레이와 시스템에 어떻게 반영되어 있는지 같이 봐야겠다.
